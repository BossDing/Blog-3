# Blog
一个基于SpringBoot+MyBatis+Shiro整合的博客系统

后台:采用SpringBoot+MyBatis,数据库采用的Mysql,数据库的脚本文件自带。

*系统集成了Shiro,关于RBAC权限模型的设计开发将在接下来完成。*

前端:SemanticUI+Jquery,
